# Evolution3d
物理演算と進化の再現を利用した実験です。

##概要
物理演算を用いて、疑似的な生態系を再現したものです。

捕食者(黒)と被食者(緑)の二種類が存在します。
彼らはそれぞれ、体と動きの遺伝子を持っています。
一定距離まで近づけば、被食者は食べられ、捕食者は増殖します。
一定時間で捕食者は飢え死に、被食者は増殖します。

いわゆる、遺伝的アルゴリズムではありません。
